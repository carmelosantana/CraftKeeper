The open-source Minecraft server control plane.

Capable, organized, protective. This kit defines the palette, type, tokens and core components CraftKeeper is built from — technical, but understandable. Switch theme and accent above; every element re-themes live.

Logo & marks

The mark reads as an organized 3×3 grid — a crafting grid, an inventory, a binder of ordered knowledge. Structural, not decorative. Never reproduce Minecraft textures or trademarks.
